Android APK

The installable Android package is produced on your computer with Capacitor and Android Studio (this screen cannot download an APK by itself).

After a debug build, your APK is usually here:
Forest Logger App\android\app\build\outputs\apk\debug\app-debug.apk
  1. Install Node.js and Android Studio (with Android SDK).
  2. Open a terminal in the app folder, run: npm install
  3. First time only (no android folder): npx cap add android
  4. Then: npm run cap:sync
  5. Then: npx cap open android
  6. In Android Studio: Build → Build Bundle(s) / APK(s) → Build APK(s).

Help & Guide
1. Getting Started ▶
Tap New Survey on the Tools page to begin. Once a session is created, select any of the 8 tool modules to record field data. Data auto-saves to your offline database.
2. Map & GPS ▶
Drop custom waypoints, track your live path, and switch between Satellite, Terrain, and Hybrid layers. Auto-logs altitude, precision, and UTM formatting.
3. Quadrat Sampling ▶
Define quadrat dimensions. Enter species lists with Abundance, DBH (cm), Height (m), and Phenology stages.
4. Belt Transect ▶
Set heading bearing and total length. Add interval intercepts documenting canopy cover and discrete distance.
5. Environment ▶
One-tap auto-fill pulls live Temperature, Humidity, and Altitude. Manually log Slope, Aspect, Canopy Cover, and Soil type.
6. Disturbance & CBI ▶
Record human/fire/grazing disturbances (severity 0-5). Calculate Composite Burn Index via Strata grading matrices.
7. Media (Photos & Voice) ▶
Capture localized site images (auto-compressed) and record acoustic environmental voice notes directly to the survey.
8. Analytics Engine ▶
Calculates Shannon-Wiener Diversity (H'), Simpson’s Index (D), Evenness (J'), and Importance Value Index (IVI). Displays DBH histograms and species accumulation.
